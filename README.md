# Queue Application for call center project

This project was made to study how the communication between client and server is made in a call center, and how does the queue works for the calls.
Its emphasys is in backend, so there is just a basic frontend to show how it works and be interactive. The main focus in in the data structures, memory, and algorithms
that had to be made for it.
